CheckboxRange.js

jQuery plugin to select checkbox range by simple connect two checkboxes by drag&drop.

Version: 1.0.0
Dependencies: jQuery â‰¥ 1.8
License: MIT
Repository: git.rafaelpawlos.com/checkboxrange
Author: Rafael Pawlos

Basic usage: 

$('.container-class').checkboxrange();
Options
path: 'any' | 'vertical' | 'horizontal'
noStyle: false | true
lineOffsetTop: 10
lineOffsetLeft: 10
onSelectEnd: function () {}

Any path default {path:any}

Vertical path {path:vertical}

Horizontal path {path:horizontal}
